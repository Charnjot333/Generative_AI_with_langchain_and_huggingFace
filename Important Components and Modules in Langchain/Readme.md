# LangChain Learning Repository

This repository contains my hands-on learning and implementation of the **LangChain Framework** for building **Retrieval-Augmented Generation (RAG)** applications.

## Topics Covered

- LangChain Ecosystem
- LangSmith
- LangServe
- LangChain Community
- Prompt Engineering
- Data Ingestion using Document Loaders
- Text Splitting
  - Recursive Character Text Splitter
  - Character Text Splitter
- Embeddings
  - OpenAI Embeddings
  - Hugging Face Embeddings
  - Ollama Embeddings
- Vector Databases
  - FAISS
  - ChromaDB
- Retriever
- Semantic Search
- Retrieval-Augmented Generation (RAG)

## Technologies Used

- Python
- LangChain
- OpenAI
- Hugging Face
- Ollama
- FAISS
- ChromaDB

## 🎯 Purpose

The purpose of this repository is to understand and implement the core building blocks of LangChain used in modern RAG applications. It covers the complete workflow, including document loading, text splitting, embedding generation, vector storage, and semantic retrieval.

This repository serves as my learning reference before building end-to-end LLM and RAG-based AI applications.
